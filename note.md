Requirement:
- Php: php >= 8.0
- Mysql: mysql >= 8.0
- Git: git version >= 2.x

Install XAMPP 
- Window: https://quantrimang.com/xampp-cach-de-dang-de-cai-dat-webserver-trong-windows-83995
- Ubuntu: https://topthuthuat.com/cach-cai-dat-xampp-tren-ubuntu/

###### Run Project
Clone code project:
```bash
git clone https://github.com/hopdt1404/SPA_IotAgri
cd SPA_IotAgri
npm install
composer install
npm run watch
php artisan serve
```
